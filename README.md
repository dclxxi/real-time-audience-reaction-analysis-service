# time-audience-reaction-analysis-service

## 📌 Summary
> 강연자의 연설 효과를 높이기 위해 청중의 반응을 실시간으로 분석해 강연자에게 제공하고, 강연 후에는 분석 결과 차트와 ChatGPT 피드백을 통해 객관적인 강의 평가를 제공하는 웹 서비스입니다.

프로젝트 기간 : `2023.12 ~ 2024.01`
